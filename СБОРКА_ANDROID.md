# Сборка Android APK — Голосовой помощник

## Шаг 1 — Установи Node.js

1. Открой https://nodejs.org
2. Скачай версию **LTS** (зелёная кнопка)
3. Установи, везде нажимай "Next"
4. Проверь в командной строке:
   ```
   node --version
   ```

## Шаг 2 — Установи Android Studio

1. Открой https://developer.android.com/studio
2. Скачай и установи Android Studio
3. При первом запуске пройди мастер установки — он сам скачает Android SDK
4. После установки запомни путь к SDK (обычно `C:\Users\ИМЯ\AppData\Local\Android\Sdk`)

## Шаг 3 — Настрой переменную ANDROID_HOME

1. Открой **Параметры Windows → Система → О системе → Дополнительные параметры системы**
2. Нажми **Переменные среды**
3. В разделе "Переменные пользователя" нажми **Создать**:
   - Имя: `ANDROID_HOME`
   - Значение: `C:\Users\ИМЯ\AppData\Local\Android\Sdk`
4. Найди переменную `Path`, нажми **Изменить**, добавь строку:
   ```
   %ANDROID_HOME%\platform-tools
   ```
5. Нажми OK везде, **перезапусти компьютер**

## Шаг 4 — Скачай код проекта

Через интерфейс poehali.dev: **Скачать → Скачать код** — получишь ZIP-архив.

Распакуй в удобную папку, например `C:\Projects\voice-assistant\`

## Шаг 5 — Установи зависимости и собери

Открой командную строку (Win+R → `cmd`) в папке проекта:

```bash
# Установи зависимости
npm install

# Собери веб-версию
npm run build

# Добавь Android платформу (только первый раз!)
npx cap add android

# Синхронизируй с Android
npx cap sync android

# Открой в Android Studio
npx cap open android
```

## Шаг 6 — Собери APK в Android Studio

1. Android Studio откроется автоматически с проектом
2. Подожди пока внизу завершится индексация (может занять 3-5 минут)
3. Меню **Build → Build Bundle(s) / APK(s) → Build APK(s)**
4. Готовый APK будет по пути:
   ```
   android\app\build\outputs\apk\debug\app-debug.apk
   ```

## Шаг 7 — Установи на телефон

1. На телефоне: **Настройки → Безопасность → Неизвестные источники** (разреши)
2. Перекинь `app-debug.apk` на телефон (через USB или Telegram себе)
3. Открой файл на телефоне — установится приложение

---

## Разрешение микрофона в фоне (важно!)

После установки:
1. Зайди в **Настройки телефона → Приложения → Голосовой помощник**
2. **Разрешения → Микрофон** → выбери **"Разрешить всегда"**
3. **Батарея** → выбери **"Без ограничений"**

Теперь приложение будет слушать команды даже при свёрнутом экране.
